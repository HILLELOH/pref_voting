Coalition Formation Demo

Given agents with differing policy positions, this algorithm finds a majority coalition that agrees on an AI-generated compromise. Read the paper →

How it works: Each iteration, the two agents with the most differing positions are selected. GPT generates candidate compromise sentences between them. Each agent votes to accept if the proposal is closer to their ideal than the current status quo (measured by cosine dissimilarity of sentence embeddings). If a majority accepts, the algorithm halts with the winning sentence and coalition.
